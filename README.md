# Giraffa Javascript SDK

A javascript SDK to interact with a Giraffa node based on [Substrate](https://github.com/paritytech/substrate)

## Getting Started

### Installing

```
yarn bootstrap
```

### Building

To build everything
```
yarn build
```

To build a certain package
```
yarn build packages/package-name
```
